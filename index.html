¿Que es FlexBox?

Flexbox es un modelo de diseño en CSS que permite a los desarrolladores crear diseños flexibles y responsivos en páginas web. El modelo Flexbox se basa en la idea de que un contenedor (llamado "flex container") puede contener uno o más elementos (llamados "flex items"), y que estos elementos pueden ser distribuidos automáticamente dentro del contenedor según ciertas reglas.

En el modelo Flexbox, se utiliza el eje principal y el eje secundario para distribuir los elementos dentro del contenedor. El eje principal es la dirección principal en la que los elementos se distribuyen, mientras que el eje secundario es la dirección perpendicular al eje principal. La dirección del eje principal se define mediante la propiedad flex-direction.

Algunos ejemplos de FlexBox

Las propiedades más importantes de Flexbox incluyen:

align-items

que define cómo se distribuyen los elementos a lo largo del eje secundario. Se pueden elegir cinco opciones: flex-start, flex-end, center, baseline y stretch

justify-content

que define cómo se distribuyen los elementos a lo largo del eje principal. Se pueden elegir cinco opciones: flex-start, flex-end, center, space-between y space-around.

flex-direction

que define la dirección en la que los elementos se distribuyen dentro del contenedor. Se pueden elegir cuatro opciones: row, row-reverse, column y column-reverse.

display: flex

que convierte un contenedor en un contenedor flexible
